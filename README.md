PHP Database Web Application

A simple web application built using PHP, MySQL, HTML, and CSS.
This project demonstrates how to connect PHP with a database and display dynamic content using a clean frontend design.

Features
PHP backend with MySQL database
HTML & CSS frontend
CRUD operations (Create, Read, Update, Delete)
Secure database connection
Responsive UI
Easy to configure and deploy

Technologies Used
PHP – Server-side scripting
MySQL – Database
HTML5 – Structure
CSS3 – Styling
Apache (XAMPP) – Local server
